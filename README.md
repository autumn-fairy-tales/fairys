# fairys

# 开发

```bash
# 安装
yarn install

# 监听组件变化 
cd packages/fairys-core & yarn watch

# 启动预览网站
cd website & yarn start
```

# 样式约定
表单组件默认高度大、默认、小依次为 `36px`、`32px`、`28px`
圆角 `2px`
主题色 `#165dff`
边框色 `#d9d9d9`
